# Estructura-de-Datos

**Entregas de Proyecto d Estrucutra de Datos del semestre 2025-30**
Esta guía está diseñada para ejecutarse en sistemas operativos basados en Linux, tales como:
  - Ubuntu
  - Debian
  - Linux Mint
  - Kali Linux
  - Pop!_OS


**Instalacion de compilador de C++**
  sudo apt update
  sudo apt install g++

**Comandos de compilación**
g++ -std=c++17 *.cxx -o programa

**Comando de ejecución**
./programa
